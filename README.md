# Racktar

Racktar is a storage rack tracking system designed for organizations that need to keep track of items across multiple storage locations.

## Features

- **Rack Management** - Create and organize storage racks by location
- **Item Tracking** - Add, edit, and track items stored in each rack
- **QR Code Support** - Generate QR codes for racks and scan them with your phone for quick access
- **Search** - Find items across all racks instantly
- **Temporary Removal** - Mark items as temporarily removed without deleting them
- **User Roles** - Control who can view, edit, or manage the system
- **Audit Trail** - Track all changes made to racks and items

## Use Cases

- Warehouse inventory management
- Office supply tracking
- Workshop tool organization
- Club/organization equipment management
- Any scenario where items are stored across multiple locations
